Stylesheets

Stylesheets are documents written in a language called CSS that change the look of a website. They're referenced in html with a <link> element. In the head of your HTML, you should have a line of code that looks like this:

<link href="style.css rel="stylesheet" type="text/css" />

The style.css links to the file in your project by that name. It's empty right now, so it doesn't do anything. We'll use it later when we learn to write CSS. In the meantime, we can take advantage of the fact that HTML lets you to link to anything on the internet, not just what's on your own site. That means you can use someone else's stylesheet (if they let you, of course!)
